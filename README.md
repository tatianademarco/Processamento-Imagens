# Manipulação de Imagens com C# e Windows Forms

Este projeto é uma aplicação Windows Forms implementada como trabalho final da disciplina de Processamento de Imagens do curso de Ciência da Computação, esta aplicação traz a manipulação de imagens usando C#, fornecendo dezenas de funcionalidades de edição de imagens, sendo possível carregar e, após tratar as imagens com filtragens diversas além de outras técnicas de manipulação avançada, salvar a imagem editada em seu computador.
